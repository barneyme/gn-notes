gn - Get Notes

Zero-dependency markdown notes sync for GitHub and Koofr.

Setup Instructions

Linux

1. Download gn-linux.zip
2. unzip gn-linux.zip
3. chmod +x gn-linux
4. sudo mv gn-linux /usr/local/bin/gn
5. Run: gn

macOS Apple Silicon (M1/M2/M3)

1. Download gn-macos-arm.zip
2. unzip gn-macos-arm.zip
3. chmod +x gn-macos-arm
4. sudo mv gn-macos-arm /usr/local/bin/gn
5. First run: right-click gn in Finder and choose Open, then click Open
6. Run: gn

macOS Intel

1. Download gn-macos-intel.zip
2. unzip gn-macos-intel.zip
3. chmod +x gn-macos-intel
4. sudo mv gn-macos-intel /usr/local/bin/gn
5. First run: right-click gn in Finder and choose Open
6. Run: gn

NetBSD

1. Download gn-netbsd.zip
2. unzip gn-netbsd.zip
3. chmod +x gn-netbsd
4. doas mv gn-netbsd /usr/local/bin/gn
5. Run: gn

Windows

1. Download gn-windows-amd64.exe.zip
2. Unzip to get gn-windows-amd64.exe
3. Rename to gn.exe
4. Move gn.exe to C:\Tools\ or any folder in your PATH
5. Open Windows Terminal and run: gn

Windows PowerShell Script (alternative)

1. Download gn.ps1 and gn.bat
2. Put both in same folder, add folder to PATH
3. Run: gn

Source Code

1. Download gn.go
2. Requires Go 1.22+: go build -o gn gn.go

Config file is created at ~/gn/gn.conf on first run.
